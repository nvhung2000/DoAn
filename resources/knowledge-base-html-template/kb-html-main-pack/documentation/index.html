How to change flickr ID!

Flickr Photos Widget is fully functional widget. You just have to give your flickr ID and thats it. To change flickr ID first you have to know your flickr ID. To get flickr ID against your required flickr profile visit http://idgettr.com/. Once you get your ID, you have to open custom.js file and enter your ID at highlight place in given screenshot.

E) Live Search Function - top

Knowledge Base HTML Template offers live search function using jQuery LiveSearch Plugin. It is AJax based function which sends request to search.php file and get processed date from it without loading the page. For now it is getting fixed contents from search.php file. It can be attached to any server side script easily.
